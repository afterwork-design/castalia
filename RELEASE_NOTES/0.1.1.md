# Release v0.1.1

发布说明：修复 CI 构建失败并做若干改进。

主要变更：
- 修复 `.eslintrc.json` 配置格式（将规则移入 `rules`）。
- 将 `typescript` 升级到 `^4.9.5`，解决 `@types` 中新语法导致的类型错误。
- 修复若干导致构建被 ESLint 拦截的问题（为 `Image` 添加 `alt`、为 `target="_blank"` 添加 `rel="noreferrer"`、移除重复 JSX props）。
- 将 GitHub Actions 中的 Node 版本从 `14.x` 升级到 `18.x`，以兼容 `package-lock.json` 的 lockfile v3。

如何验证：
1. CI 在 `main` 分支触发时应能成功通过 `npm ci` 与 `npm run build` 并部署到 gh-pages（如果已配置）。
2. 本地复现：
```bash
npm install
npm run build
```
